2.9. Sync

Sync Dialog collect all operation related with remote repository to one dialog. Sync Dialog included push, pull, fetch, remote update, send patch ...

TortoiseGit → Sync... .

Sync Dialog will show.

2.9.1. Branch

Local Branch: The source branch which will push\pull to\from other repository.

Remote Branch: The remote branch of the other repository.

2.9.2. Destination

Remote URL: Choose remote repository or input remote repository URL.

Manage Add new remote name.

2.9.3. Options

Force Force Overwrite Existing Branch(May discard changes)

Autoload putty key Autoload putty key when push or pull
