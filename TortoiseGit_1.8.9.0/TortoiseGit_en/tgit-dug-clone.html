2.3. Clone Repository

This section talks about how to clone a git repository from an existing repository. Cloning a git repository is very simple. At an empty directory, just use the explorer context menu and select Git Clone... .

Clone Dialog will show.

Url: Input repository URL address, which you will clone from. You can click Browse to browse directory.

Directory:Input your local directory, which you will clone to. You can click Browse to browse directory.

If you check the Load Putty Key checkbox, clone will auto load putty key file with Pageant. You can click ... to browse putty key file.

Clone will checkout current HEAD to work space automatically.

Git clone supports http, git and ssh protocol. Section 2.35.3, “Network Settings” shows how to choose SSH client. OpenSSH, PLink or TortoiseGitPlink.

You can find more information at Section G.3.21, “git-clone(1)”
