2.15. Reference Log

Manage reflog information. TortoiseGit → RefLog You have to hold the Shift key while right clicking on the item in the explorer list pane (right pane) in order to see this in the extended context menu.

Figure 2.24. RefLog Dialog


The RefLog can be used to restore deleted commits or HEAD positions (e.g. when you deleted a branch which was HEAD some time ago).
